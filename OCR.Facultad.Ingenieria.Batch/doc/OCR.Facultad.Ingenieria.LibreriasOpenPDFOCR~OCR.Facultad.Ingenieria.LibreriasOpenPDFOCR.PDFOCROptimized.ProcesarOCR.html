PDFOCROptimized.ProcesarOCR Method

Realiza el Reconocimiento Óptico de Carateres(OCR), para un documento TIF.

El resultado es un par de documentos(PDF Y TXT), con contenido UNICODE del texto extraido luego del proceso OCR.

public static int ProcesarOCR(
   string unimagesource,
   string unfolderoutput,
   bool isdeskew
);

Parameters

unimagesource
unfolderoutput
isdeskew
Si es TRUE, realiza el enderezamiento automático de las páginas del documento TIF.

Return Value

Número de páginas procesadas luego del proceso OCR.

Exceptions

Exception Type Condition
ExceptionThreadOCR Captura las excepciones disparadas por los hilos de programación.

Example

Como realizar el Reconocimiento Óptico de Caracteres de un documento TIF 

int paginas = PDFOCROptimized.ProcesarOCRPDFTXT(@"C:\prueba.tif",@"C:\prueba.pdf",@"C:\prueba.txt",true);
